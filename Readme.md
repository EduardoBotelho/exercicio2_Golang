# Exercicio 2(Exercise 2)

## Segundo exercício chamado somatório(Second exercise called Summation)
### Escreva um programa que ache o somatório de cada número de 1 até o número. O número vai ser positivo e inteiro maior que zero. Sua função é somente retornar o resultado, o qual é mostrado entre parentêses no exemplo abaixo é como você deve chagar ao resultado e isto não é parte disso, veja os exemplos de teste.
## For example (Input -> Output):

### 2 -> 3 (1 + 2)
### 8 -> 36 (1 + 2 + 3 + 4 + 5 + 6 + 7 + 8)
